# safefallFullstack

- 현재 진행상황
1. 라즈베리파이에 yolo11 올리기 성공
2. yolo11을 통한 낙상감지 성공
3. 사고 영상을 백엔드가 위치한 로컬 환경에 저장 성공 및 썸네일 생성 확인
4. 라즈베리파이의 영상을 백엔드에 전송 성공
5. 백엔드와 프론트엔드 통신 성공
6. 회원가입 페이지 제작성공 및 회원가입 성공 확인

- 필요 작업
1. 영상이 DB에 저장되는지 확인 필요
2. 알람 발생 확인 필요
3. DB에 저장된 영상정보와 프론트간의 실시간 연동 확인 필요
4. 회원가입 후 자동 로그인 가능 여부 확인 필요

- 추후 작업
1. 모델의 추가학습을 통한 인식률 개선
2. 영상의 전송 딜레이가 현재 10초 내외 이므로 추후 개선
3. 프론트로 전송되는 영상이 프레임이 떨어지는 것으로 확인. 추후 개선

## Front
각 코드의 역할은 주석으로 설명을 작성해 놓음.

## Back
각 코드의 설명은 introduceBackEnd라는 텍스트 파일로 작성해 놓음

## RASP
현재 사용된 코드들의 복사본
첨부된 사진은 코드들이 저장되어있는 디렉토리 모습을 캡쳐해 놓은 것
전원을 키면 자동으로 코드가 실행되도록 셋팅해 놓음
